Specifications:
----------------------

https://cducoin.org

• ASICS friendly

• 51% attacks resistant

• Adopts Segwit transations

• Uses the POW (Proof of Work)

• Algorithm:        SHA256

• Block Time:       6 seconds

• Block Reward:     6 CDU

• Difficulty:       16 blocks

• Halving:          700 mil blocks (133 years)

• Ticker:           CDU

• Block size:       4mb

• P2P Port:         8077

• RPC Port:         8076

• Max supply:       8,399,999,992

• POW Confirmations:   26
